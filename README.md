# Latent Dynamics of Prediction-Market Price Paths

The thesis site for **Juan Mediavilla** — MSc Machine Learning, University College London,
supervised by Prof. Philip Treleaven. Served from GitHub Pages at
[juanmediavilla.com](https://juanmediavilla.com).

An absorption-aware intensity model of high-frequency prediction-market price formation, built on a
purpose-built, gap-audited, millisecond-resolution order-book corpus of Polymarket.

## Structure

```text
index.html          Landing — abstract, contributions, and links to the two views
framework/          Abstract and the eight-chapter framework
slides/             The web slide deck
slides/figs/        Experiment figures (PNG)
CNAME               juanmediavilla.com
.nojekyll           Serve as-is; no Jekyll processing
```

Every page is static: one embedded stylesheet each, no web fonts, no CDN, no third-party requests.
All assets are served from this repository.
